seth fox

pilot wave

  • release date: 1/1/15
  • gifjam, ableton (Free trial)
  • cover art made in pixlr

this EP was made using gifjam, by tim thompson, which used to be available as a webpage. it took input in a variety of patterns by sampling pixel color changes on a gif image. the output was almost like a randomized midi file, i did a lot of editing (cutting, pasting, deleting sections, changing notes and son on). it's named after david bohm's interpretation of quantum mechanics.
