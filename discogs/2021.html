uninhabitable zones

  • release date: 8/9/21
  • ableton, yamaha tenori-on

i wrote these myself (no midi generators involved). it seemed like it was time to return to that, after several generator-written projects. this was my first album using ableton 10. previously i had been using the cheap version, the wavetable synth is what finally made me save up for it.
